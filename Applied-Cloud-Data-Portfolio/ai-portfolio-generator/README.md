# 🚀 Portfolio.AI: AI-Powered Personal Portfolio Generator

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen.svg)](https://personal-portfolio-generator-34827481300.europe-west3.run.app/)

## 📌 Overview
**Portfolio.AI** is a dynamic, next-generation web application that leverages generative AI to instantly transform unstructured professional backgrounds, CVs, or LinkedIn summaries into beautifully designed, ready-to-deploy single-page portfolios. 

This project was built end-to-end and deployed via **Google AI Studio** and **Google Cloud Run** to showcase the practical integration of Large Language Models (LLMs) in modern web and product development.

## ✨ Features
* **Intelligent Parsing:** Utilizes **Gemini 3.5 Flash** to analyze and structure career timelines, technical skills, and educational backgrounds from raw text inputs.
* **Instant Generation:** Generates a complete, responsive dark-themed portfolio website in seconds without manual template tweaking.
* **Interactive Design Workspace:** Clean, modern UI with sections for Hero (Bio), Experience, Education, and Skills.
* **Production-Ready Hosting:** Containerized and hosted on Google Cloud Run for fast, reliable, and scalable global access.

## 🛠️ Tech Stack
* **AI Engine:** Google Gemini 3.5 Flash API
* **Development & Prototyping:** Google AI Studio
* **Deployment & Cloud Infrastructure:** Google Cloud Run (GCP)

## 💡 How It Works
1. Navigate to the live link.
2. Paste your unstructured CV, LinkedIn profile text, or a raw professional paragraph into the Background Parser.
3. Click **Generate My Website**.
4. The AI engine dynamically processes the data and constructs a tailored, high-contrast HTML layout instantly.
