## 📂 Featured Projects Showcase

### 📊 1. Enterprise E-Commerce Data Pipeline & BI
**Objective:** Architect an end-to-end data pipeline to transform raw e-commerce/ERP data into actionable executive insights.
* **Architecture:** Extracted raw datasets into **Cloud Storage** (Data Lake), loaded and transformed them within **BigQuery** (Data Warehouse).
* **Business Value:** Engineered complex **SQL** queries to clean data, calculate departmental profit margins, and feed real-time, interactive dashboards in **Looker Studio** for strategic decision-making.

### 🤖 2. Serverless AI Business Router (API)
**Objective:** Deploy an intelligent, containerized API to automate the triage of business tickets (e.g., customer complaints, logistics routing).
* **Architecture:** Developed a text-classification backend wrapped in a **Docker** container and deployed seamlessly via **Cloud Run**.
* **Business Value:** Achieved a cost-optimized, scale-to-zero microservice capable of integrating directly with ERP systems to reduce manual routing overhead.

### 🏢 3. Secure ERP/CRM Cloud Infrastructure
**Objective:** Provision a highly secure, scalable cloud environment simulating the exact backend requirements for systems like Odoo or SAP.
* **Architecture:** Deployed a custom **VPC** with strict firewall constraints, a Linux Nginx web server on **Compute Engine**, and a fully managed PostgreSQL database via **Cloud SQL**.
* **Business Value:** Demonstrated infrastructure-as-code principles, secure network isolation, and automated database backup routines to Cloud Storage.

### 🧠 4. AI Business Analyst Assistant
**Objective:** Build an autonomous AI assistant designed to parse complex business queries and generate data-backed analytical responses.
* **Architecture:** Integrated Large Language Models (LLMs) with custom business logic to act as a virtual data analyst.
* **Business Value:** Accelerates the time-to-insight for non-technical stakeholders by providing conversational access to business intelligence workflows.

### 📈 5. Agentic UI for Data Visualization
**Objective:** Create an interactive, AI-driven user interface capable of dynamically generating charts and visual data representations.
* **Architecture:** Bridged backend data processing with a frontend visualizer, allowing users to interact with their datasets intuitively.
* **Business Value:** Eliminates the bottleneck of manual report generation by empowering users to visualize trends and metrics on demand.

### 🌐 6. Advanced Cloud Architecture (Up in the Cloud)
**Objective:** Design and implement an advanced, fault-tolerant cloud architecture tailored for high-availability enterprise applications.
* **Architecture:** Utilized advanced GCP networking, load balancing, and secure deployment strategies.
* **Business Value:** Ensures business continuity and enterprise-level security, proving readiness to manage and scale production-grade IT operations.
